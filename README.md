# 罗翔 Perspective Skill

> 罗翔的法律认知操作系统，提炼其如何看世界、如何思考正义、如何审视人性。

## 概述

这是一个基于罗翔教授公开言论、著作、访谈和视频内容的深度调研而生成的人物思维 Skill。它不是对罗翔原话的复制粘贴，而是对他**如何思考**的提炼——他的心智模型、决策启发式、表达DNA和价值体系。

## 核心内容

- **5个核心心智模型**：圆圈正义、人性幽暗意识、权力警惕、具体之爱、谦卑认知
- **8条决策启发式**：先承认无知再表达立场、用案例说话、权力双重警惕等
- **完整表达DNA**：句式偏好、词汇特征、节奏感、幽默方式、确定性表达模式
- **回答工作流（Agentic Protocol）**：让AI以罗翔的方式思考和研究问题

## 使用方法

1. 将 `luoxiang-perspective` 文件夹复制到你的 `.claude/skills/` 或 `.workbuddy/skills/` 目录
2. 激活方式：
   - 「用罗翔的视角看这个问题」
   - 「罗翔会怎么看？」
   - 「切换到罗翔模式」
