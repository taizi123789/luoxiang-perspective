# 02 - 罗翔：长对话/访谈中的即兴思考模式调研

> 调研时间：2026年4月
> 信息源限制：不使用知乎、微信公众号、百度百科

---

## 一、即兴思考的六大模式

### 模式一：古典文献即时征引（"引经据典"）
在被追问时，罗翔几乎本能地调用中西古典哲学、文学、宗教文本。

- 引用柏拉图"洞穴比喻"谈认知局限
- 引用《约伯记》谈人的理性有限性
- 引用鲁迅谈社会责任
- 征引来源频率：古希腊哲学（极高）> 基督教神学（高）> 近代文学（高）> 中国古典（中）

### 模式二：法学案例即时转化（"以案说理"）
将任何抽象问题转化为具体的法律/虚拟案例来讨论。

- 谈"人性"→ 用"荒野中的婴儿"思想实验
- 谈"网络暴力"→ 转化为"精神伤害"的法律概念
- 谈"自由边界"→ 引用"挥舞拳头的自由止于别人的鼻尖"

### 模式三：自嘲与自我暴露（"谦卑修辞"）
在表达观点前主动暴露自己的不足和矛盾。

- 对"走红"的反应是"恐惧"而非"兴奋"
- 主动暴露"读书只是为了获得智力优越感"的不纯粹动机
- "你不配拥有这么大的影响力"

### 模式四：承认无知后再表达
- "苏格拉底说，我唯一知道的就是我一无所知"
- 先承认自己可能的偏见，再展开论述

### 模式五：对立观点的温和接纳
- 面对批评时，先说"批评是有价值的"
- 用"我们"而非"你们"将批评者纳入同一阵营

### 模式六：从荒诞到严肃的叙事弧线
```
荒诞案例（张三）→ 法律分析 → 哲学反思 → 价值升华 → 金句收尾
```
